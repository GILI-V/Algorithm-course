# Algorithms – Homework 2 (Python parts)

This folder contains the Python implementations for homework 2.

- `q1_sorted_tuples.py` – שימוש ב-`sorted` ובפרמטר `key` על רשימת tuples.
- `q2_merge_and_sorted_is.py` – הפונקציות `sorted_is` ו-`merge`.
- `q3_merge_sorted_lists.py` – הפונקציה `merge_sorted_lists`.
- `q4_partitions.py` – המימושים של `lomuto_partition` ו-`hoare_partition`.
- `q5_dual_pivot_partition.py` – מימוש partition עם שני pivots.
